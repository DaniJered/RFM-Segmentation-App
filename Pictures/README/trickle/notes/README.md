# README AI Generator

A cinematic futuristic AI SaaS interface for a README auto-generator.

## Features
- Deep-space premium aesthetic with glassmorphism UI
- Split-screen layout (Input Workspace & Live Rendered Preview)
- Dynamic tech stack detection chips
- Real-time streaming token generation effect
- Simulated markdown parsing and rendering
- Persistent Trickle Database integration to save generated READMEs
